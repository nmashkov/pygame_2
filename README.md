# Game 2
## Описание
Разминочная задача для отработки моторного взаимодействия в паре и контроля его эффективности.
Левый и правый обследуемые с помощью своих управляющих клавиш должны удержать красный шар на вершине горы, сопротивляясь переменчивому ветру.
## Технологии
- Python 3.9.10
- Pygame 2.3.0
- Pymunk 6.4.0
